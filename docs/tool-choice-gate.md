# Tool Choice Gate

Before building anything, answer these.

1. Is the workflow already useful manually?
2. Does this need a backend?
3. Does this need Gemini-native behaviour, realtime, video, voice or multimodal testing?
4. Does this need a simple interactive tool only?
5. Does this need recurring automation across apps?
6. Is this a real build or productive procrastination?

Decision ladder:

- unclear workflow: stay manual
- Gemini-native prototype: Google AI Studio
- simple interactive internal tool: Claude Artifacts
- full-stack app with backend: Lovable
- recurring workflow across apps: Make or n8n
- code-heavy product: Claude Code, Cursor or developer support
