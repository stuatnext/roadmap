# Privacy Rules

Use this roadmap without leaking sensitive data.

Do not paste into external tools unless approved and safe:

- employee performance details
- PIP details
- CRM exports
- client data
- personal data
- compensation data
- sensitive revenue data
- private emails or Teams messages
- confidential NEXT.io strategy

Use anonymised examples by default.

For external tools, replace names with roles and replace exact numbers with ranges unless exact data is required and approved.
