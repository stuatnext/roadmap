# Operating Rules

1. One prompt equals one decision, one asset, or one workflow.
2. No dashboard unless it changes weekly behaviour.
3. No agent unless the manual workflow already works.
4. No strategy unless it creates a decision.
5. No AI comparison after the third model.
6. No asset gets built without checking what exists.
7. No plan without a kill condition.
8. No AI power-user work unless it saves time, improves quality, reduces risk, or creates revenue.
9. Chinese is a strategic track, not a hobby.
10. NEXT efficiency comes before extra ambition, because NEXT is draining the fuel.
