# Stuart One Roadmap

A private, repo-ready operating roadmap for Stuart Crowley.

This is not another strategy document. It is a task-led system for reducing NEXT.io chaos, building commercial confidence, proving Strait Up Growth, learning AI and tech properly, growing a useful network, and making Chinese a strategic Singapore asset.

## How to use it

1. Open `index.html` through GitHub Pages, or run a local web server.
2. Pick the active sprint.
3. Choose no more than 3 active tasks.
4. Use the task card to pick the right model, asset, prompt, action, success test, and kill rule.
5. Complete the Friday Review before choosing new work.

## Local use

Because the app loads JSON files, use a local server rather than double-clicking the HTML file.

```bash
python3 -m http.server 8000
```

Then open:

```text
http://localhost:8000
```

## GitHub Pages

1. Create a private GitHub repo.
2. Upload this folder.
3. Go to Settings > Pages.
4. Set source to `main` branch and root folder.
5. Keep the repo private unless you remove all personal and company-specific references.

## Safety rules

- Do not upload confidential NEXT.io data.
- Do not include client data, API keys, credentials, private emails, or exports.
- Use this as a control system, not as another planning rabbit hole.
- No automation until the manual workflow has worked for at least 2 weeks.
- No more than 3 active tasks at once.
