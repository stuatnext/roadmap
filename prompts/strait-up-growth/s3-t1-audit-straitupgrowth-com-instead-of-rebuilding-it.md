# Audit straitupgrowth.com instead of rebuilding it

**Sprint:** Sprint 3: Market Contact and Proof

**Lane:** Strait Up Growth

**Use this model or tool:** ChatGPT Project for strategy. Gemini or Perplexity for live site and market checks. Claude for copy edits.

**Use this asset:** Strait Up Growth OS: core-positioning.md, audience-map.md, differentiation.md, offer architecture. CV proof base.

## Starter prompt

```text
Audit my existing Strait Up Growth website.

Goal: improve conversion for founder-led B2B SMEs in Singapore.

Check:
1. Does the homepage state the problem sharply?
2. Does it make me credible?
3. Does it say who I help?
4. Does it say what I offer?
5. Is AI positioned as useful but not hype?
6. Is Singapore clear enough?
7. Is the CTA strong?
8. Does the voice sound like me?
9. What should not be changed?
10. What are the smallest high-impact edits?

Output: keep, change, remove, add, test. No full rebuild unless evidence demands it.
```

## Human starter action

Open the site. Write the current headline, CTA and offers. Then run the audit.

## Expected output

A conversion-focused site audit tied to your real positioning and Singapore market.

## Success test

A short edit list, not a rebuild plan.

## Friday review question

Did I improve an existing asset rather than create a new one?

## Kill rule

If the output suggests a new website before auditing the current one, reject it.
