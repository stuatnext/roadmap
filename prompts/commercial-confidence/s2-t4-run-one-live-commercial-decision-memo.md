# Run one live commercial decision memo

**Sprint:** Sprint 2: Commercial Confidence

**Lane:** Commercial Coaching

**Use this model or tool:** ChatGPT Project first. DeepSeek for critique. Claude for stakeholder adaptation.

**Use this asset:** Commercial Director Advisory Board: review a strategy, strategy teardown template, meeting prep prompts

## Starter prompt

```text
Coach me through this live commercial decision.

First ask me for missing data. Then help me produce:
1. Recommendation.
2. Evidence.
3. Assumptions.
4. Weakest point.
5. Stakeholder objections.
6. Decision needed.
7. Next action.

Keep it short enough to send to leadership. Do not give generic commercial theory.
```

## Human starter action

Pick one live decision from NEXT.io. Write the recommendation in one sentence before opening AI.

## Expected output

One live decision becomes a short, structured memo.

## Success test

One decision memo produced and used.

## Friday review question

What judgement improved this week?

## Kill rule

If the memo cannot be used in a real discussion, it is too abstract.
