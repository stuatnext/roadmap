# Create one GTM readiness gate

**Sprint:** Sprint 2: Commercial Confidence

**Lane:** GTM Discipline

**Use this model or tool:** ChatGPT Project.

**Use this asset:** Executive Growth System: GTM_Readiness_Gate.md, GTM_Readiness_Audit.md. Commercial Advisory: GTM scorecard

## Starter prompt

```text
Create a GTM readiness gate for NEXT.io launches.

It must score: product definition, ICP, value proposition, pricing, margin, capacity, deadlines, owner, CRM setup, campaign plan, sales collateral, reporting, approval status and delivery risk.

Output: red, amber, green checklist with required evidence, owner and go or no-go rule.

Constraint: must fit on one page. No theory.
```

## Human starter action

List the next launch. Score it red, amber or green. Share missing items with owners.

## Expected output

A red, amber, green gate that stops half-ready launches entering sales and marketing.

## Success test

One launch has visible readiness gaps before chaos starts.

## Friday review question

What did the gate expose before it became drama?

## Kill rule

Do not use the gate as bureaucracy. Use it as protection.
