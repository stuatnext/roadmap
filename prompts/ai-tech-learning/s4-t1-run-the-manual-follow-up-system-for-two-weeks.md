# Run the manual Follow-Up System for two weeks

**Sprint:** Sprint 4: Manual AI Workflows

**Lane:** AI Leverage

**Use this model or tool:** Claude Cowork if connected to MS365. ChatGPT Project if using pasted notes.

**Use this asset:** LLM Control Hub: ownership clarifier, meeting rescue, red team review. Executive Growth System: Outlook Teams signal rules, Daily Stuart Radar.

## Starter prompt

```text
Using this list of meetings, emails, Teams messages and notes, extract every follow-up I owe or need from others.

Return a table: owner, ask, deadline, blocked decision, risk, suggested message, escalation date, status.

Rules: do not invent, mark missing context clearly, separate my actions from other people actions, and flag anything commercially risky.
```

## Human starter action

At the end of each day, paste notes/messages into the workflow and update the tracker. Do not automate yet.

## Expected output

Evidence that the follow-up workflow saves time and reduces missed actions.

## Success test

Used 8 times across the sprint. Clear evidence of saved time or unblocked decisions.

## Friday review question

Did the manual workflow earn automation?

## Kill rule

Do not build an agent because the idea feels clever.
