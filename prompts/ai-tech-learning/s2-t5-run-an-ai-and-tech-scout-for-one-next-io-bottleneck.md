# Run an AI and tech scout for one NEXT.io bottleneck

**Sprint:** Sprint 2: Commercial Confidence

**Lane:** AI and Tech Learning

**Use this model or tool:** Gemini Deep Research or ChatGPT Deep Research. Perplexity for quick scan. ChatGPT Project for decision.

**Use this asset:** LLM Control Hub: source hierarchy, ai-tool-evaluation. Executive Growth System: current bottleneck notes.

## Starter prompt

```text
Find 5 current AI or tech solutions for this bottleneck: [CRM hygiene, follow-up tracking, sales handoff, outreach personalisation, AI governance, workflow automation, or meeting action extraction].

Prioritise official sources, recent evidence, pricing clarity, HubSpot integration, Microsoft 365 integration, Monday.com integration, Apollo integration, and realistic setup effort.

For each tool, give: use case, cost signal, setup effort, risks, integrations, data/privacy concern, and recommendation: test, monitor or ignore.

Constraint: do not include tools unless they solve my stated bottleneck.

Success test: one 7-day test selected or all tools killed with reasons.
```

## Human starter action

Pick one bottleneck only. Do not research all tools. If no tool is worth testing, document why.

## Expected output

One tool or platform selected for a 7-day test, or all options killed with reasons.

## Success test

One tool test or one clear kill decision.

## Friday review question

Did learning reduce a real NEXT or SUG problem?

## Kill rule

If you spend more than 60 minutes browsing tools, stop.
