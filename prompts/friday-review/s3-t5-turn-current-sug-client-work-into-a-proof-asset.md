# Turn current SUG client work into a proof asset

**Sprint:** Sprint 3: Market Contact and Proof

**Lane:** Proof

**Use this model or tool:** Claude for writing. ChatGPT Project for structure and critique.

**Use this asset:** Strait Up Growth OS: case study template, client delivery templates. CV proof base.

## Starter prompt

```text
Help me turn current client work into an anonymised Strait Up Growth proof asset.

Inputs I will provide: client type, starting problem, what changed, tools used, measurable result if known, qualitative result, before-after, quote if available.

Output:
1. Short case study.
2. LinkedIn-safe version.
3. Website proof block.
4. Sales call talking point.
5. Missing proof I should gather.

Constraint: no inflated claims. Mark anything unproven.
```

## Human starter action

List what changed for the client. Ask for permission or anonymise carefully.

## Expected output

One anonymised case study or before-after proof note.

## Success test

One proof block created.

## Friday review question

What proof did I create instead of claiming expertise?

## Kill rule

If the result cannot be evidenced, call it an observation, not a case study.
