# Package what worked and kill what did not

**Sprint:** Sprint 6: Package and Automate

**Lane:** Packaging

**Use this model or tool:** ChatGPT Project for review. DeepSeek for critique.

**Use this asset:** All assets. Review notes. Quality Gate. Friday reviews.

## Starter prompt

```text
Review the last 12 weeks using my roadmap evidence.

For each lane: NEXT.io control, commercial confidence, Strait Up Growth, personal brand and network, AI leverage, Chinese.

Output:
1. what worked
2. what did not
3. what to keep
4. what to kill
5. what to automate
6. what to simplify
7. next 30-day focus

Rules: be brutal, evidence-led and concise. No new strategy unless the evidence supports it.
```

## Human starter action

Gather completed tasks, notes and actual outcomes before asking AI.

## Expected output

A keep, improve, kill and automate decision across the roadmap.

## Success test

One kill list and one next focus list.

## Friday review question

What did I stop doing because it did not work?

## Kill rule

If everything is kept, the review failed.
