# Install the owner-to-owner communication rule

**Sprint:** Sprint 1: Relief and Control

**Lane:** Operating Discipline

**Use this model or tool:** ChatGPT Project for scripts. Claude for stakeholder tone.

**Use this asset:** Executive Growth System: Ownership_Map.md, Delegation_Responses.md, Leadership_Communication_Rules.md

## Starter prompt

```text
Context: I get pulled into communicating between colleagues at NEXT.io. I need to stop being human glue.

Task: Create my owner-to-owner communication rule.

Output:
1. One clear rule.
2. When I should be copied.
3. When I should not be involved.
4. Five ready-to-send responses.

Tone: calm, professional, not dismissive.

Success test: I can redirect one request this week without sounding unhelpful.
```

## Human starter action

Use this once: Please update them directly and copy me only if a decision is blocked.

## Expected output

A simple rule and response bank that routes communication back to the correct owner.

## Success test

One middleman request redirected.

## Friday review question

Where did I stop being human glue?

## Kill rule

Do not explain the philosophy every time. Use the rule.
