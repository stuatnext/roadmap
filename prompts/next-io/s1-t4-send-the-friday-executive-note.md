# Send the Friday Executive Note

**Sprint:** Sprint 1: Relief and Control

**Lane:** Executive Communication

**Use this model or tool:** Claude Project for writing. ChatGPT Project for tightening.

**Use this asset:** Executive Growth System: Weekly_Executive_Script.md, Pierre_Weekly_Pack.md, Upward_Management_Playbook.md

## Starter prompt

```text
Context: I need Pierre and Ana to see commercial control without me writing long defensive updates.

Task: Draft my Friday Executive Note.

Output format, under 200 words:
1. Progress.
2. Risk.
3. Decision needed.
4. Blocked owner.
5. Next week focus.

Style: BLUF, concise, executive-ready.

Constraints: no waffle, no blame, no emotional language.

Success test: leadership can see what matters and where decisions are needed.
```

## Human starter action

Write five bullets yourself: progress, risk, decision, blocker, next focus. Then sharpen with AI.

## Expected output

A short note showing progress, risk, decision needed, blocker and next focus.

## Success test

One executive note sent or templated.

## Friday review question

Did I make leadership visible this week?

## Kill rule

If it reads like a diary, cut it back to decisions and risks.
