# Set up the next 30-day sprint without hardcoded dates

**Sprint:** Sprint 6: Package and Automate

**Lane:** Next Quarter

**Use this model or tool:** ChatGPT Project for synthesis. Human final.

**Use this asset:** Roadmap, review notes, completed task data.

## Starter prompt

```text
Based on my completed tasks and review notes, set the next 30-day sprint.

Choose:
1. one NEXT.io control task
2. one commercial confidence task
3. one SUG or market task
4. one AI or tech learning task
5. one Chinese task

For each: why, model, asset, prompt, human action, success test and kill rule.

Constraint: no more than 5 active tasks. No hardcoded dates.
```

## Human starter action

Pick the next active sprint in the Command tab. Choose three tasks, not ten.

## Expected output

A new current sprint selected with 3 active tasks and a Friday review routine.

## Success test

Next sprint configured without creating a new system.

## Friday review question

What is the next smallest useful focus?

## Kill rule

If the next sprint has more than five active tasks, cut it.
