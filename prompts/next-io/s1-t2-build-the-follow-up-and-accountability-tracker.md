# Build the Follow-Up and Accountability Tracker

**Sprint:** Sprint 1: Relief and Control

**Lane:** NEXT.io Control

**Use this model or tool:** ChatGPT Project for design. Claude Cowork if pulling from Outlook, Teams or Monday.com.

**Use this asset:** Executive Growth System: Ownership_Map.md, Decision_Rights_Register.md, Daily_Triage.md, Outlook_Teams_Signal_Rules.md

## Starter prompt

```text
Context: I lose time chasing people at NEXT.io for decisions, approvals and missing information. I need a simple tracker that stops me chasing from memory.

Task: Design the tracker and chase messages.

Output format:
1. Tracker fields.
2. Daily 5-minute use routine.
3. Polite chase.
4. Firmer chase.
5. Escalation.
6. Rules for when to escalate.

Constraints: usable in Monday.com or a spreadsheet. No new app. No over-engineering.

Success test: I use it at least four times this week.
```

## Human starter action

Create columns: owner, ask, deadline, blocked decision, risk, last chase, next chase, escalation date, status. Add the five most urgent open asks.

## Expected output

One tracker showing who owes what, by when, what is blocked and what message to send next.

## Success test

Tracker used 4 times. One blocked decision becomes visible.

## Friday review question

What did I stop carrying in my head?

## Kill rule

If it takes more than 5 minutes a day, simplify it.
